Core Mission
This utility exists to hold judicial actors accountable to the true Rule of Law through Deterministic Logic. The jury operates under bylaws the group has established by itself, including the group’s process for adding an AI juror (or possibly deleting one). All processes, decisions, changes are publicly viewable
